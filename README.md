# uk-census-data

In July 2025 I runa short project to "Vibe Code" a data presentation app. The application pulled data 2021 census data from the API of the UK Office of National Statistics, and presented these in graphs. The code is available in [GitHub](https://github.com/tommyarmstrong/UKCensus) and the app is available [here](https://ukcensus.tommyarmstrong.uk).
